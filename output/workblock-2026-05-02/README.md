## CLI patches landed during this run (uncommitted in TutorialVid repo)

- `record/index.ts` — pass `storageState` to `newContext()` so localStorage JWT loads (was cookies-only).
- `record/runner.ts` — sleep before each action to honor `t_ms` timeline (videos used to terminate when actions completed).
- `compose/render.ts` — entry points to `dist/compose-entry.js` (avoids webpack TS extension issues); spins up tiny static HTTP server (Remotion 4.x rejects `file://`).
- `compose/index.ts` — `Math.max(audio_ms, scene_target_ms)` so video duration not capped by short audio.
- `compose/timeline.ts` — bbox-based zoom anchor + Ken Burns drift + blend across adjacent zooms.
- `compose/zoom-state.ts` (new) — `getCurrentZoom()` + `transformBboxToScreen()` so highlight overlays can render in screen-space and stay pixel-perfect at any zoom.
- `compose/components/Spotlight.tsx` (new) — SVG mask cutout, transforms bbox to screen-space.
- `compose/components/HighlightFrame.tsx` (new) — pulsing glow ring, screen-space bbox.
- `compose/components/Callout.tsx` — 8 anchor positions (4 sides + 4 corners), fade+slide, gradient bg.
- `tts/gemini.ts` — wrap raw PCM in WAV header (Gemini returns L16 PCM, not MP3); plaintext prompt instead of SSML (preview model truncates on em-dashes / parens); model name `gemini-2.5-flash-preview-tts`.
